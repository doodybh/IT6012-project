2. Design and Code Justifications

Impact of Design and Choice of Colors on Delivered Content

The chosen color palette focuses on blue (#007b9e) and its variations, which is commonly associated with trust, stability, and health in the medical field. This choice was deliberate to reinforce the websiteâ€™s purpose as a trustworthy Health Portal.

  • Blue/Teal Tones: Used for headers, navigation, and primary call-to-action buttons (e.g., Book Appointment). This ensures crucial interactive elements stand out and convey professionalism.

  • White/Light Blue Backgrounds: The light background (#e9f5f9 and #f0f9ff) maximizes content legibility and reduces eye strain, which is vital for pages containing medical information or detailed articles.

  • Font Choice: A clear, sans-serif font (Arial/Helvetica) was selected to optimize readability across all devices, ensuring the delivered medical and health content is easily consumed by all users.

Impact of HTML, CSS, JS, and Bootstrap Code on Maintainability of Website

Our coding structure prioritized clarity, modularity, and adherence to best practices to ensure the website is easy to maintain and scale for future development:

  • CSS Structure: The external `Style.css` file uses clear comment dividers to organize styles by section (e.g., NAVBAR, DOCTOR CARDS, PAGE HEADERS). This modular approach makes it quick for any developer to locate and modify specific styling rules.

  • Bootstrap Integration: Using Bootstrap (CSS and JS bundle) allowed for rapid development of responsive components (Grid system, Navbar, Modals) and ensures consistent cross-browser compatibility with minimal custom CSS.

  • HTML/JS Separation: Functional logic is strictly placed in separate JavaScript files, keeping the HTML clean and focusing on semantic structure. Event handlers (`onclick`) are used directly on elements for simplicity in this project scope.

  • Naming Conventions: Classes (e.g., .navbar-custom, .info-card, .doctor-card) are named clearly and consistently to reflect their function, greatly improving code readability.
